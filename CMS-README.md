# BEGZ Law Consulting — 4 хэлтэй админ CMS

Энэ хувилбарт сайт дээр `/admin/` хаягаар орох Decap CMS админ хуудас нэмэгдсэн. Админ хэсгээс дараах контентыг нэмж, засаж, нууж/нийтэлж болно.

## Засварлах үндсэн хэсгүүд

1. **Манай баг** — багийн гишүүн, албан тушаал, мэргэшил, танилцуулга, зураг.
2. **Түүхэн товчоо** — байгууллагын он цагийн замнал, milestone.
3. **Хууль зүйн мэдээлэл** — хууль зүйн мэдээллийн картууд, зөвлөмжийн заалтууд.
4. **Шүүхийн шийдвэрийн тойм мэдээ** — кейс, шүүхийн шийдвэрийн тойм, амжилттай шийдвэрлэсэн маргааны дэлгэрэнгүй.
5. **Түгээмэл асуулт** — асуулт, хариулт.

## 4 хэлний бүтэц

Админ дээр харагдах бүх гол текст дараах 4 талбартай:

- Монгол (`mn`) — заавал бөглөнө.
- English (`en`) — сонголтоор.
- 中文 (`zh`) — сонголтоор.
- 한국어 (`ko`) — сонголтоор.

Жишээ бүтэц:

```json
"title": {
  "mn": "Монгол гарчиг",
  "en": "English title",
  "zh": "中文标题",
  "ko": "한국어 제목"
}
```

Сайт дээр хэрэглэгч хэл сонгоход тухайн хэлний текстийг автоматаар сонгож харуулна. Англи/Хятад/Солонгос талбар хоосон байвал Монгол эхээр fallback хийж харагдана.

## Админ нэвтрэх тохиргоо

Netlify дээр байршуулсны дараа:

1. Netlify → Site settings → Identity → **Enable Identity**.
2. Identity → Services → **Enable Git Gateway**.
3. Identity → Invite users хэсгээс админ хэрэглэгчийн имэйлээ урьж бүртгэнэ.
4. Сайт дээр `/admin/` хаягаар орж нэвтэрнэ.

## Контент хадгалагдах файлууд

- `content/team.json`
- `content/history.json`
- `content/legal-guide.json`
- `content/cases.json`
- `content/faq.json`

Админ дээр хадгалах үед эдгээр JSON файлууд Git repo-д commit болж хадгалагдана.

## Нийтлэх / нуух

Бичлэг бүр дээр **Нийтлэх эсэх** гэсэн toggle байгаа. Унтраавал тухайн бичлэг сайт дээр харагдахгүй. Устгахгүйгээр түр нуухад хэрэглэнэ.

## Дараалал

Бичлэг бүр дээр **Дараалал** талбар байгаа. Бага тоотой бичлэг түрүүлж харагдана. Жишээ нь 1, 2, 3.

## Анхаарах зүйл

Энэ CMS нь шинэ Монгол текстийг өөрөө машин орчуулахгүй. Админ хэрэглэгч 4 хэлний орчуулгыг өөрөө оруулна. Ингэснээр хууль зүйн нэр томьёо алдаагүй, мэргэжлийн байдлаар харагдана.
